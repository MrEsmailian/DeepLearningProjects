# Deep Learning Projects – Fall 2025

This repository contains a collection of deep learning projects completed as part of the
**Deep Learning** course offered in **Fall 2025** at
**[Amirkabir University of Technology (Tehran Polytechnic)](https://aut.ac.ir/)**
under the supervision of **Dr. Safabakhsh**.

All projects are implemented in **Python** using the **PyTorch** framework and are
organised as standalone Jupyter notebooks with accompanying problem statements,
dataset generators, and output results.

## Projects Overview

### 1. Feed‑Forward Neural Networks
**Objective:** Build and analyse single‑layer and multi‑layer perceptrons for
non‑linear binary classification.

**Datasets:** Synthetic “crescents” and “spirals” datasets generated by `data_gen.py`.

**Key Topics:**
- Sigmoid vs. ReLU activations
- Effect of hidden layer size on separability
- Hyper‑parameter tuning (learning rate, number of neurons, regularisation)
- Decision boundary visualisation

**Notebook:** [`Deep_Learning_Project1.ipynb`](https://github.com/MrEsmailian/DeepLearningProjects/blob/main/P01.Feed.Forward.Neural.Networks/Deep_Learning_Project1.ipynb)

---

### 2. Convolutional Neural Networks
**Objective:** Implement and evaluate CNN architectures for image classification
on Fashion‑MNIST.

**Dataset:** [Fashion‑MNIST](https://www.kaggle.com/datasets/zalando-research/fashionmnist)
(grayscale 28×28 images, 10 classes).

**Key Topics:**
- Convolutional, pooling, and fully‑connected layers
- Batch normalisation and dropout regularisation
- Comparison of different architectures (varying depth and width)

**Notebook:** [`Deep_Learning_Project2.ipynb`](https://github.com/MrEsmailian/DeepLearningProjects/blob/main/P02.Convolutional.Neural.Networks/Deep_Learning_Project2.ipynb)

---

### 3. Transfer Learning
**Objective:** Leverage pre‑trained models (e.g., ResNet, VGG) for CIFAR‑10
image classification and compare with training from scratch.

**Dataset:** [CIFAR‑10](https://www.cs.toronto.edu/~kriz/cifar.html)
(32×32 colour images, 10 classes).

**Key Topics:**
- Feature extraction vs. fine‑tuning
- Impact of freezing/unfreezing layers
- Comparison of different pre‑trained backbones

**Notebook:** [`Deep_Learning_Project3.ipynb`](https://github.com/MrEsmailian/DeepLearningProjects/blob/main/P03.Transfer.Learning/Deep_Learning_Project3.ipynb)

---

### 4. Time‑Series Forecasting (RNN)
**Objective:** Design recurrent neural networks (LSTM/GRU) for sequence
prediction tasks.

**Dataset:** [Electricity Load Diagrams 2011-2014](https://archive.ics.uci.edu/ml/datasets/electricityloaddiagrams20112014).

**Key Topics:**
- RNN, LSTM, and GRU cells
- Sequence‑to‑sequence modelling
- Hyper‑parameter search (sequence length, hidden size, number of layers)

**Notebook:** [`Deep_Learning_Project4.ipynb`](https://github.com/MrEsmailian/DeepLearningProjects/blob/main/P04.Time.Series.Forecasting/Deep_Learning_Project4.ipynb)

---

### 5. Generative Adversarial Networks
**Objective:** Train a GAN to generate realistic Fashion‑MNIST images.

**Dataset:** [Fashion‑MNIST (same as Project 2)](https://www.kaggle.com/datasets/zalando-research/fashionmnist).

**Key Topics:**
- Generator and discriminator architectures
- Adversarial loss formulations
- Training stability and mode collapse mitigation

**Notebook:** [`Deep_Learnning_project5.ipynb`](https://github.com/MrEsmailian/DeepLearningProjects/blob/main/P05.Generative.Adversarial.Networks/Deep_Learnning_project5.ipynb)

---

### 6. Large Language Models (PEFT)
**Objective:** Apply parameter‑efficient fine‑tuning (PEFT) techniques to a
pre‑trained transformer model.

**Dataset:** [Wiki QA](https://huggingface.co/datasets/microsoft/wiki_qa).

**Key Topics:**
- LoRA (Low‑Rank Adaptation) and other PEFT methods
- Fine‑tuning vs. full training trade‑offs
- Evaluation of language generation quality

**Notebook:** [`Deep_Learnning_project6.ipynb`](https://github.com/MrEsmailian/DeepLearningProjects/blob/main/P06.Large.Language.Models/Deep_Learnning_project6.ipynb)

## Setup & Usage

1. **Clone the repository**
   ```bash
   git clone https://github.com/MrEsmailian/DeepLearningProjects.git
   cd DeepLearningProjects
   ```

2. **Create a virtual environment (optional)**
   ```bash
   python -m venv venv
   source venv/bin/activate   # Linux/Mac
   venv\Scripts\activate      # Windows
   ```

3. **Install dependencies**
   ```bash
   pip install torch torchvision jupyter numpy matplotlib scikit-learn
   # For Project 6:
   pip install transformers peft datasets
   ```

4. **Launch the notebook of interest**
   ```bash
   jupyter notebook P01.Feed.Forward.Neural.Networks/Deep_Learning_Project1.ipynb
   ```

Each notebook contains the full workflow: data loading, model definition,
training loop, evaluation, and result visualisation. Simply run all cells to
reproduce the results.
